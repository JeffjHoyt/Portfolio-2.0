Order summary component project

The Challenge

This challenge was to build out this order summary card component and get it looking as close to the design as possible. The catch was that I wasn't given actual dimensions of any of the components so I had to recreate the UX designs into code without a blueprint

The Solution

Since I was only given jpegs as my blueprint then my first challenge was to translate these images into pixel sizes in order to replicate them with code. So, I first used Adobe XD to import the images and scale them into data that could be used to design the project.
